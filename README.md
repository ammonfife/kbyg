## üìñ Usage

### Basic Flow

1. **Setup (First Time):**
   - Click extension icon ‚Üí Open side panel
   - Add Gemini API key ([Get one free](https://aistudio.google.com/app/apikey))
   - Fill in your company info

2. **Capture Leads:**
   - Visit any conference website (e.g., Utah Tech Week)
   - Click extension icon
   - Click **"Analyze Event"**
   - Review extracted companies and contacts

3. **Sync to Dashboard:**
   - Data automatically saves to GTM Intelligence Hub
   - Visit [Web Dashboard](https://easy-event-planner.lovable.app)
   - All captured companies appear instantly!
